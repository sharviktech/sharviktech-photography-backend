Angular frontend scaffold instructions

This folder contains sample Angular components and services. To create the Angular application run:

ng new sharvik-frontend --routing --style=scss
cd sharvik-frontend

Then copy the files from this folder into the generated Angular project (`src/app` etc.) and run `npm install` and `ng serve`.

Update `environment.ts` to set the API base URL to the backend (e.g., `http://localhost:5000/api`).
